Imprime um vetor bidimensional de caracteres em formato de palavras cruzadas.

Esta função percorre o array answers, e para cada entrada, imprime os caracteres um a um, separados por espaços, simulando a visualização de uma palavra cruzada. O índice da palavra (1 a 10) é mostrado antes de cada linha.

Parameters
answersArray 2D de caracteres representando as respostas (com '_' ou letras).
wordSelectArray 2D de índices que indicam quais palavras estão sendo usadas a partir do wordList.
wordListLista completa de palavras usadas para obter os comprimentos corretos.
