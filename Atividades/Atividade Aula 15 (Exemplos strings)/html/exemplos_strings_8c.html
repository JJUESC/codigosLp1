Separa e armazena palavras de uma string em um array 2D.

Esta função percorre a string input e separa as palavras usando o espaço (' ') como delimitador. Cada palavra encontrada é armazenada como uma string em uma linha do array word.

Parameters
inputA string de entrada contendo palavras separadas por espaço.
wordArray 2D onde cada palavra será armazenada em uma linha. Cada linha deve ser capaz de armazenar até 31 caracteres + '\0'.
Returns
O índice da última palavra armazenada (ou seja, o total de palavras - 1).
